# Overview

This project was developed to strengthen my skills as a software engineer in data analysis, data cleaning, and data visualization using real-world datasets. My goal was to build a structured workflow that takes raw data, processes it, and extracts meaningful insights that can support decision-making.

The dataset used in this project is a movie dataset (mymoviedb.csv) containing information such as movie titles, release dates, popularity scores, average ratings, and original language. This type of dataset is commonly used in entertainment analytics and recommendation systems.

Dataset source: https://www.kaggle.com/datasets/sanyamdhadiwal2302/netflix-data

The purpose of this software is to analyze movie data and answer key questions such as:

What are the highest-rated movies?
Is there a relationship between popularity and ratings?
How has movie production changed over time?
Which languages dominate the dataset?

By answering these questions, the project demonstrates how data analysis can uncover patterns and trends in large datasets.

# Data Analysis Results

The following questions were explored during the analysis:
* What are the top 10 highest-rated movies?
The dataset was sorted by average rating, and the top 10 movies were identified. These represent the highest-quality films based on user ratings.
* Is there a relationship between popularity and ratings?
A correlation analysis showed a weak positive relationship, meaning popular movies tend to have slightly higher ratings, but the relationship is not strong.
* How many movies are released each year?
The analysis showed an increasing trend in movie releases over time, especially in recent years.
* What is the average movie rating?
The overall average rating gives a baseline for comparing movie performance across the dataset.
Which movies are more popular than average?
Movies with popularity scores above the dataset average were filtered, highlighting films that stand out in audience attention.
* Which languages are most common?
The dataset revealed that a few languages dominate, with English being the most frequent.
